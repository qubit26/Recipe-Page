# Recipe Page — Omelette Demo

## English

A small demo project to showcase frontend skills. This repository contains a single-page application that displays an omelette recipe with ingredients, instructions, and a simple layout designed for clarity and responsiveness.

Key points:
- Purpose: demo of frontend abilities (layout, component structure, styling).
- Content: recipe for an omelette (ingredients + step-by-step instructions).
- UX: responsive, accessible basics, clear typographic hierarchy.

Technologies:
- React (Vite)
- Tailwind CSS
- Vite for fast development and bundling

Quick start:
1. Install dependencies:
    - npm install
2. Run development server:
    - npm run dev
3. Build for production:
    - npm run build

## Español

Una pequeña demostración para mostrar habilidades frontend. Este repositorio contiene una aplicación de una sola página que muestra la receta de un omelette con ingredientes, instrucciones y un diseño sencillo, claro y responsivo.

Puntos clave:
- Propósito: demostrar habilidades frontend (maquetación, estructura de componentes, estilos).
- Contenido: receta de un omelette (ingredientes + pasos).
- UX: responsivo, consideraciones básicas de accesibilidad y jerarquía tipográfica clara.

Tecnologías:
- React (Vite)
- Tailwind CSS
- Vite para desarrollo rápido y empaquetado

Inicio rápido:
1. Instalar dependencias:
    - npm install
2. Ejecutar servidor de desarrollo:
    - npm run dev
3. Generar versión de producción:
    - npm run build

Resumen:
Un proyecto demostrativo y autocontenido pensado para presentar capacidad en React, Vite y Tailwind CSS, mostrando una receta simple y una interfaz limpia.
